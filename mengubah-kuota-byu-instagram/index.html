Cara Mengubah Kuota by.U Instagram Menjadi Kuota Reguler

By.U adalah provider digital pertama yang ada di Indonesia, jika kalian ingin kartu yang tidak ada batas masa aktifnya, by.U adalah jawabannya. by.U menyediakan paket yang berbeda-beda, salah satunya yaitu paket khusus untuk instagram. Kuota untuk instagram itu ternyata bisa diubah menjadi kuota reguler loh.

Tutorial Diupdate Juni 2020

Hal yang dibutuhkan

  1. Aplikasi Wireguard - bisa ditemukan di Google Playstore
  2. Config Wireguard - bisa dibuat gratis di berbagai web

Buat Config Wireguard

Setelah kamu menginstall Wireguard, sekarang yang dibutuhkan adalah config wireguard. Config tersebut bisa kalian buat di berbagai web penyedia layanan vpn gratis seperti fastssh,jagoanssh, oceanssh, dan lainnya.

Kali ini saya beri petunjuk bagaimana cara membuat config wireguard di fastssh.com

Pertama buka website ini dan pilih server yang masih tersedia, saya sarankan gunakan server indonesia atau singapura.

Lalu buat buat config atau akun wireguardnya, nantinya setelah mengisi username dan password config,akan muncul tampilan seperti dibawah ini, kemudian tekan download, untuk mengunduh confignya.

Config Wireguard selesai dibuat, sekarang saatnya mengubah config tersebut agar bisa digunakan untuk mengubah kuota instagram by.U ke kuota reguler.

Mengubah Config Wireguard

Buka aplikai wireguard kamu, lalu tambah konfig yang sudah dibuat dibagian bawah app yang bertuliskan import from file or archive, atau kalau mau simpel scan saja barcode yang tertera seperti foto diatas

Setelah itu edit config tersebut agar support untuk mengubah kuota instagram menjadi reguler dengan cara menambahkan bug host di bagian endpoint.
